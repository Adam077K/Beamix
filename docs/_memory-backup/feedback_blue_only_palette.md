---
name: Blue-only color palette for marketing
description: All marketing visuals must use blue shades only — no multi-color score tiers, no orange/red/green/amber
type: feedback
---

Marketing cards and showcase visuals must use ONLY blue with different gradients and shades. No multi-color score system (cyan/green/amber/red).

**Why:** The multi-color approach looks like "AI slop" to the user. Blue-only is cleaner and more on-brand (#3370FF primary).

**How to apply:** When building marketing visuals, charts, score indicators, progress bars — use variations of blue (#3370FF, lighter/darker shades, blue gradients). Do NOT use the product's score-tier colors (cyan #06B6D4, green #10B981, amber #F59E0B, red #EF4444) in marketing materials.
