User is very hands-on with design. Key preferences discovered during marketing showcase sessions:

**Loves:**
- Glass/frosted effects on light backgrounds (backdrop-blur-md, rgba white)
- Figma-style multiplayer cursors representing AI agents
- Clean white backgrounds with subtle dot grids
- Engine logos (real brand logos, not letter squares)
- Scattered/organic card layouts (not rigid grids)
- Business-friendly language ("Getting you mentioned" not "Optimizing brand visibility")
- Animated progress bars and score rings

**Hates:**
- Arbitrary bright colors that don't match the data (use engine palette)
- Technical/developer language in marketing materials
- Rigid grid layouts when organic scatter looks better
- Components that are too small or narrow (Group D was too narrow as a list)

**Process:** User wants to be involved — reviews in browser, gives pixel-level feedback (move cursor down, flip it, move card to middle). Expect iterative position adjustments.

**Why:** User is building a Framer marketing website and screenshots these components directly. They must look premium enough to compete with Attio, Linear, Vercel.

**How to apply:** Always create both light and dark versions of major components. Default to engine color palette. Use business-friendly copy. Expect position tweaks.
