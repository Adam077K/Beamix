This is the starting point for all design work. Vercel-quality look + Notion-quality feel (warm, content-focused, rich editing).

**How to apply:** 
- All designer agents (design-lead, frontend-developer, design-critic) must reference this before any UI work
- Quality target: Vercel's polish + Notion's warmth
- Brand tokens override Vercel's palette (Beamix blue #3370FF, Inter/InterDisplay, etc.)
- The design.md provides component patterns, spacing, typography scale, motion guidelines
