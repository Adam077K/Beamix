**7 new modules to add beyond Frame 5 v1:**
1. **AI Engine Test Harness** — anyone (customer or non) submits arbitrary URL/content → Beamix tests against all 19 surfaces. Free viral acquisition surface.
2. **Industry Knowledge Graphs** — vertical-specific datasets (one per vertical) accumulating cross-customer learning. Crown-jewel data moat.
3. **Citation Vault** — every citation Beamix has ever found for a customer, searchable, exportable, permanent record. Per-customer switching cost.
4. **Predictive Layer** — Citation Predictor, score forecasting, competitor-move forecasting, AI-engine-update impact alerts.
5. **Workflow Builder** — power users (Yossi types) build custom agent workflows via DAG-style UI. "If competitor X publishes Y, run Z, notify on Slack."
6. **Reputation Layer** — beyond AI engines. Reviews, social mentions, news, brand mentions across the web. TAM expansion.
7. **Content Studio** — customer co-authors content WITH Beamix (Cursor-for-content pattern). Hybrid AI-human creation. Low-trust on-ramp for users uncomfortable with full autonomy.
