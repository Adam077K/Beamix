# Beamix

> GEO (Generative Engine Optimization) Platform for SMBs — scan, diagnose, and fix AI search visibility.

## Repository

**Project repo:** [https://github.com/Adam077K/Beamix](https://github.com/Adam077K/Beamix)

```bash
git clone https://github.com/Adam077K/Beamix.git
cd Beamix
```

## Project Structure

| Path | Purpose |
|------|---------|
| `saas-platform/` | Next.js app — main product (landing, dashboard, agents) |
| `client/` | (if present) Additional client apps |
| `.planning/` | PRD, architecture, competitive research, specs |
| `agent-memory/` | Agent memory and research context |
| `GSA-Vibe-Startup-Kit/` | GSA toolkit (separate repo, do not modify) |

## Key Documents

- **PRD:** `.planning/PRD.md`
- **Architecture:** `.planning/TECHNICAL_ARCHITECTURE.md`
- **Strategy:** `.planning/STRATEGIC_FOUNDATION.md`
- **Competitive Research:** `.planning/COMPETITIVE_RESEARCH.md`

## Quick Start

```bash
cd saas-platform
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
