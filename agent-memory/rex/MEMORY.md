## Key Standing Patterns

### Source Rating Rules (always apply)
- G2/Trustpilot/Reddit = ★ signal, not proof — quote for pain language, not as verified facts
- Official docs, SEC filings, annual reports = ★★★ primary
- Reputable journalism, analyst reports = ★★ secondary
- Never present a single source as definitive

### Project Context
- Product: GEO Platform (Beamix) — SMB-focused, $250-600/month
- Target: SMBs 5-50 employees in services, professional services, hospitality
- Core differentiator: Track + Diagnose + Generate content + Re-track (no competitor does all 4)
- Primary competitors: HiGoodie (direct), SEMrush (incumbent), Writesonic (content-first), NP Digital (weak)

### GEO Signals That Matter Most (priority order)
1. Citation density in authoritative sources — highest LLM-agnostic signal
2. Structured data / schema markup — especially LocalBusiness + FAQ schema
3. Direct-answer content formatting — H2/H3 as questions, answer-first paragraphs
4. Entity disambiguation and co-occurrence — consistent naming everywhere
5. Review signals across platforms — volume + recency + responses
